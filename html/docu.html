Octobre 2021

AP3 - GSB Gestion de visites

Détails

Atelier Pro. - Projet "GSB Visites"

Résumé:
GSB Gestion de Visites, qui est la seconde variante des projets GSB proposés au 3ème semestre, est un sujet qui doit être réalisé dans le cadre du BTS SIO. Ce projet fera partie d'un bouquet de quatre projets réalisés par l'étudiant au cours de ses deux années de BTS.
Application:
L’application devra permettre aux visiteurs médicaux (représentant un laboratoire) de gérer leurs visites au travers d'une application web. Les visiteurs pourront gérer leurs rapports (ajouter, modifier, rechercher par date) et gérer les médecins qui leurs sont associés, ainsi que les rapports concernant chacun d'eux.
Cette application devra utiliser un langage de programmation orienté objet, et devra adopter une technologie différente de celles utilisées lors de l'AP 2 et 4.
Technologies:
Afin de mener ce projet à bien, j'ai utilisé le Framework "Laravel", ce dernier exploitant le language Orienté Objet "PHP", et prenant forme autour d'une architecture Modèle-Vue-Contrôleur.
Laravel permet, entre autres, de fournir de nombreuses fonctions favorisant la cybersécurité du site web. Le choix d'utiliser un Framwrok permet à la fois de changer d'environnement de développement et de rencontrer de nouvelles méthodes et contraintes de développement, mais également d'étudier et mettre en pratique une technologie très largement utilisée dans le monde professionnel.
Tâches mises en oeuvre:
- Réaliser un site web respectant l'architecture MVC (Modèle Vue Contrôleur), et le tout Orientée Objet.
- L'application devra comprendre des opérations de lecture, création, modification et suppression (CRUD) de données
- Le projet devra correspondre au cahier des charges, et est ainsi plus restrictif que pouvait l'être l'AP 2

AP4 - GSB Gestion de médecins

Détails

Atelier Pro. - Projet "GSB Médecins"

Ce stage en milieu professionnel au sein de la banque française BNP Paribas fût réalisé dans le secteur de Valmy à Montreuil (93100).
J'ai travaillé sur l'application web "MesAvantages", une application en production permettant à des entreprises ou écoles partenaires BNP Paribas de créer leur propre "mini-site" afin de tenir compte de leurs partenariats avec BNP.
La problématique principale fût de migrer la base de données en production sur MySQL vers une instance PostgreSQL.

Activités effectuées durant ce stage:
- Mise en place d'un environnement de developpement via Docker (conteneurisation)
- Migration de base de données MySQL vers une instance PostgreSQL via PgLoader
- Configuration et maintenance d'une instance PostgreSQL
- Mise en place d'une configuration de serveyr web (Apache et Nginx)
- Rédaction d'une documentations utilisateur et technique résumant l'ensemble des processus pour la migration
- Mener à bien des tickets Jira dédiés à une application web Laravel
- Déploiment de l'application via Kubernetes et Helm
