Stage 2ème année - BNP Paribas

Ce stage en milieu professionnel au sein de la banque française BNP Paribas fût réalisé dans le secteur de Valmy à Montreuil (93100).
J'ai travaillé sur l'application web "MesAvantages", une application en production permettant à des entreprises ou écoles partenaires BNP Paribas de créer leur propre "mini-site" afin de tenir compte de leurs partenariats avec BNP.
La problématique principale fût de migrer la base de données en production sur MySQL vers une instance PostgreSQL.

Activités effectuées durant ce stage:
- Mise en place d'un environnement de développement via Docker (conteneurisation)
- Migration de base de données MySQL vers une instance PostgreSQL via PgLoader
- Configuration et maintenance d'une instance PostgreSQL
- Mise en place d'une configuration de serveur web (Apache et Nginx)
- Rédaction d'une documentation utilisateur et technique résumant l'ensemble des processus pour la migration
- Mener à bien des tickets Jira dédiés à une application web Laravel
- Déploiement de l'application via Kubernetes et Helm

Stage 1ère année - Graphiplast

Graphiplast est une entreprise spécialisée dans le commerce de gros. Elle réalise des présentoirs ou autres objets mettant en avant des produits que proposent les clients.
L'objectif principal était de créer un site web permettant de renforcer la présence de l'entreprise sur internet. Le site devait inclure une interface administrateur, afin que le gérant de l'entreprise puisse le modifier à sa guise.

Activités effectuées durant ce stage:
- Réalisation d'un site web en PHP Orienté Objet suivant une architecture MVC
- Mise en ligne du site web et amélioration du référencement
- Mise en place d'une instance MySQL et d'un cluster MongoDB
- Réalisation d'une interface administrateur contenant un CRUD pour chaque élément du site lié à l'instance MySQL
- Gestion d'upload et de suppression d'images
- Réalisation d'un second site web au travers d'un CMS
