Memento

Présentation de la page de Memento

Cette page regroupe l'ensemble des balises html utilisées pour faire ce site
Le code CSS est lui commenté, principalement dans le fichier "header.css" et "main.css", car ils regroupent la majeure partie du code constituant ce site.

Balise Utilité
<html> Balise ouvrant le document HTML
<head> Partie utile a l'initialisation de scripts, ou pour relier une page HTML à son CSS, ou encore pour spécifier certaines informations relatives à la page (titre, favicon...)
<link> Balise permettant de relier un autre document, principalement une feuille CSS
<script> Balise permettant d'associer à la page un script
<meta> Balise permettant d'associer à la page certaines propriétés, comme une image de couverture ou l'encodage
<header> Partie supérieure du site
<nav> Navigation, souvent inclue dans un header. Permet a l'utilisateur de naviguer de page en page
<ul> Balise représentant une liste
<li> Chaque point composant la liste
<body> Coeur de la page, regroupant la majeure partie des éléments composant la page
<section> Balise permettant de délimiter une section précise
<main> Section essentielle, la plus importante
<div> Balise permettant d'englober plusieurs composants
<p> Paragraphe
<h1 - h6> Zone de texte, les chiffres définissent l'épaisseur du texte par ordre décroissant
<a> Lien ou redirections
<img> Balise permettant l'insertion d'une image
<button> Boutton
<form> Formulaire
<input> Zone avec laquelle l'utilisateur interragit, pour écrire du texte principalement
<textarea> Large zone de texte que l'utilisateur peut renseigner
<table> Tableau
<thead> Partie supérieure de la table, utilisée pour regrouper les catégories
<tbody> Corp de la table
<tr> Ligne d'un tableau
<td> Petite case composant la ligne
<th> Petite case avec un texte mis en gras
</br> Balise permettant de sauter une ligne
